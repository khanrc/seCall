# v0.10.0 Release Notes

## Highlights

- `wiki update --review` now supports `claude`, `codex`, `haiku`, `ollama`, `lmstudio`, and legacy `anthropic` review backends.
- Added `[wiki].review_backend` and CLI `--review-backend`.
- Review prompts now enforce strict JSON more aggressively for local backends.
- `secall config set` preserves user comments via `toml_edit`.
- Added [LLM Configuration Reference](../reference/llm-config.md).
